# Monitoring Scripts

Скрипты для мониторинга системных ресурсов и производительности.

## Основные скрипты

- `system_monitor.py` - Мониторинг CPU, памяти, сети, дисков

## Использование

```bash
# Мониторинг системы во время нагрузки
python system_monitor.py
```

## Особенности

- Реальное время мониторинга ресурсов
- Графики производительности
- Детальные отчеты по использованию
