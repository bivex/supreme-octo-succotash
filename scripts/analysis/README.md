# Analysis Scripts

Скрипты для анализа производительности, профилирования и оптимизации базы данных.

## Основные скрипты

- `postgres_query_analysis_final.py` - Полный анализ PostgreSQL с рекомендациями
- `query_analyzer.py` - Детальный анализатор запросов
- `simple_query_analysis.py` - Быстрый анализ состояния БД
- `analyze_profile.py` - Анализ профилирования Python кода

## Использование

```bash
# Полный анализ БД
python postgres_query_analysis_final.py

# Анализ конкретных запросов
python query_analyzer.py
```
