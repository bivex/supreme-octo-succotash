# Setup Scripts

Скрипты для настройки среды разработки и развертывания.

## Основные скрипты

- `setup_dev.bat` - Настройка среды разработки
- `restarter.bat` - Перезапуск сервера разработки

## Использование

```batch
# Настройка среды разработки
setup_dev.bat

# Перезапуск сервера
restarter.bat
```
